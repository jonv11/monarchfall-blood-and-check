# AI Prompt Crafting Guidelines

## Purpose
Provide a modular guide for crafting reliable prompts for AI coding agents working in MFBC.

## Scope
Applies to prompt files and agent instructions used with VS Code and related automation in this repository.

## When to Use
Use this index to choose the specific module(s) needed for a task and load them explicitly in prompts.

If you are writing prompts for agent behavior, also review [Agent Rules](../core/agent-rules.md).
